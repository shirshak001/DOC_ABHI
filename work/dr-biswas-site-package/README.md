# Dr. Abhijit Biswas Premium Website

Production-ready Next.js 15 personal brand website for a senior medical professional.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Notes

- EmailJS is wired with demo IDs in `components/premium-doctor-site.tsx`; replace them with real service, template, and public key values before launch.
- Update clinic phone, email, address, and domain in `constants/site.tsx`.
- The supplied doctor portrait is stored at `public/images/dr-abhijit-biswas.png`.
